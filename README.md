# Portfolio Website - Er. Mohammad Umair Shah

A professional, animated portfolio website showcasing the work, skills, and achievements of Er. Mohammad Umair Shah, a Civil Engineer and founder of Keashur Engineer.

## 📋 Features

- **Animated Hero Section** - Eye-catching gradient background with smooth animations
- **Professional Layout** - Clean, modern design with glassmorphism cards
- **Responsive Design** - Mobile-friendly layout that works on all devices
- **Smooth Transitions** - Hover effects and animations throughout
- **Complete Profile** - About, Education, Skills, Experience, and Contact sections

## 🎨 Sections Included

1. **Hero Section** - Name, title, and brief introduction with animated gradient
2. **About Me** - Professional background and expertise
3. **Education & Achievements** - Degree, GATE qualification, and content creation
4. **Technical Skills** - Key engineering competencies
5. **Experience & Projects** - Internship details and research work
6. **Contact** - Email, phone, LinkedIn, and location

## 🚀 How to Use

1. Open `index.html` in any modern web browser
2. The site will load with animated elements
3. Scroll through sections to explore the portfolio
4. Hover over cards to see interactive effects

## 📱 Customization

You can easily customize:
- **Colors** - Modify the gradient values in the CSS
- **Content** - Update text in each section
- **Fonts** - Change from Poppins to another Google Font
- **Animations** - Adjust timing and effects in the `@keyframes`
